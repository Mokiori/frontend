Base structure:
```
..
└── components
    ├── <kebab-case-component-name>
    │   ├── index.jsx
    │   └── style.(scoped).scss (optional)
    └── utils (optional)
        ├── index.jsx
        │   └── ...
        └── ...
```

Component must be named in PascalCase style:
```typescript jsx
const ComponentName = () => {}
```
